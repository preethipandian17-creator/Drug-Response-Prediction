# Drug-Response-Prediction

Drug Response Prediction

Project Overview

This project focuses on predicting the type of drug that should be prescribed to a patient based on their medical attributes. Using machine learning, we train a model on patient data and classify the prescribed drug into one of five categories (Drug A, Drug B, Drug C, Drug X, or Drug Y).
